## position

| 属性值 | 名称 | 脱离文档流 | 基准点 | 说明 |

| static | 正常定位 | 否 | 无 | 默认值，忽略top/left/right/bottom/z-index属性 |
| relative | 相对定位 | 否 | 自身位置 | |
| absolute | 绝对定位 | 是 | 最近非static父节点 | |
| fixed | 固定定位 | 是 | 视窗 | |
| sticky | 粘性布局 | 否 |  | |
